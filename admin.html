🎨 디자인 기본 설정

사이트 전체에 적용되는 색상과 이름을 변경합니다. 색상을 바꾸면 버튼, 링크, 강조 텍스트 등 모든 포인트 색이 함께 바뀝니다.

🏷️ 사이트 기본 정보
브라우저 탭과 검색 결과에 표시되는 사이트 이름입니다. 예: HessEdU — 영어회화 전문 교육 플랫폼
🎨 포인트 컬러 사이트 전체 적용
버튼 색상, 강조 텍스트, 링크 색상 등 사이트 전체의 포인트 색상입니다. 색상 박스를 클릭하거나, 위에 #으로 시작하는 색상 코드를 직접 입력하세요.
변경 후 반드시 상단의 💾 저장하기를 눌러주세요.
저장 후 index.html을 새로고침(F5)하면 변경 사항이 반영됩니다.

🏠 첫 화면 (히어로)

사이트를 열었을 때 가장 먼저 보이는 전체 화면 섹션입니다. 브랜드명, 메인 제목, 버튼을 수정할 수 있습니다.

📍 변경 내용은 사이트 맨 위 전체 화면 영역에 표시됩니다.
✍️ 히어로 텍스트 편집
메인 제목 위에 작게 표시되는 브랜드 설명입니다. 예: HessEdU · 영어회화
화면 중앙의 큰 제목 중 첫 번째 줄입니다. 짧고 임팩트 있는 문구를 사용하세요.
두 번째 줄은 기울임꼴(이탤릭)로 표시됩니다. 첫 줄을 강조하는 보완 문구를 쓰세요.
히어로 섹션의 시작 버튼 텍스트입니다. 클릭하면 코스 섹션으로 이동합니다.

✏️ 한 줄 소개 & 통계

히어로 아래에 나오는 굵은 한 줄 소개 문구와, 수치 4개(학습 콘텐츠, 교육 경력, 수강생, 만족도)를 편집합니다.

📍 한 줄 소개는 히어로 바로 아래 큰 문장으로, 통계는 그 아래 숫자 4개 영역에 표시됩니다.
✏️ 한 줄 소개 문구
히어로 아래에 크게 표시되는 한 줄 슬로건입니다. 사이트의 핵심 가치를 간결하게 표현하세요. 이탤릭 강조는 문구 중간에 자동으로 적용됩니다.
📊 통계 숫자 4개 순서대로 표시
숫자와 설명을 각각 입력하세요. 숫자는 "400+" 처럼 단위 기호도 포함해서 입력하면 됩니다.

📚 코스 소개

사이트의 영어회화 코스 카드 4개를 편집합니다. 각 코스의 아이콘, 이름, 설명, 콘텐츠 수량, 세부 주제 4개를 수정할 수 있습니다.

📍 코스 카드는 "영어회화 코스" 섹션의 4개 카드에 표시됩니다.

👩‍🏫 강사 소개

강사 프로필 사진, 이름, 소개글, 전문 분야 태그, 자격/경력 4줄, 이메일 주소를 편집합니다.

📍 강사 소개는 "About" 섹션에 표시됩니다.

강사 소개 섹션의 제목 "Meet ○○○"에 사용되는 이름입니다.

강사 소개 섹션과 가격 정책 섹션에 표시되는 이메일 주소입니다.

강사 이름 아래에 뱃지처럼 표시되는 전문 분야입니다. 예: 영어회화, 비즈니스 영어, 발음 교정, 동시통역

💬 수강생 후기

수강생 후기 카드 3개를 편집합니다. 이름, 수강 분야, 변화/성과, 후기 내용을 수정할 수 있습니다.

🔗 GitHub 연동 설정

GitHub 토큰을 입력하면 저장하기 버튼을 누를 때마다 설정이 GitHub에 자동 업로드됩니다.
이 기능을 사용하면 어느 기기, 어느 브라우저에서 접속해도 index.html에 동일한 내용이 반영됩니다.

ℹ️ GitHub 연동 없이도 이 컴퓨터의 이 브라우저에서는 정상 작동합니다. 다른 기기에서도 반영되려면 아래 토큰을 설정하세요.
🔑 GitHub 개인 토큰 (Personal Access Token)
GitHub Personal Access Token을 입력하세요. 토큰은 이 브라우저에만 안전하게 저장되며 GitHub로 전송될 때만 사용됩니다.
토큰이 올바른지 확인합니다. 성공하면 녹색 메시지가 표시됩니다.
📋 GitHub 토큰 발급 방법
1 GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic) 접속
2 Note에 이름 입력 (예: HessEdU Admin)
3 Expiration: No expiration (만료 없음) 선택
4 repo 권한 체크박스에 ✅ 체크
5 Generate token 클릭 → 생성된 토큰을 복사해서 위 입력창에 붙여넣기

⚠️ 주의: 토큰은 한 번만 표시됩니다. 창을 닫기 전에 반드시 복사하세요.
💡 작동 원리
저장하기 버튼을 누르면 → GitHub의 hessedu_config.json 파일에 설정이 저장됩니다.
index.html 페이지를 열 때 → GitHub에서 이 파일을 자동으로 불러옵니다.

✅ 한 번 설정하면 어디서나 — 스마트폰, 다른 컴퓨터, 다른 브라우저 — 모두 동일하게 반영됩니다.
✅ GitHub Pages 사이트(https://thereisjoy.github.io/HessedEnglish/)에도 반영됩니다.

⚙️ 초기화 & 도움말

관리자 페이지 사용 방법과 전체 초기화 기능입니다.

📖 사용 방법
1 왼쪽 메뉴에서 수정하고 싶은 섹션을 선택합니다.
2 내용을 입력하거나 수정합니다.
3 상단 오른쪽의 💾 저장하기 버튼을 클릭합니다.
4 index.html 파일을 브라우저에서 열고 F5(새로고침)을 누르면 변경 내용이 반영됩니다.

💡 중요한 점:
· 저장하기 버튼을 눌러야만 변경 내용이 저장됩니다.
· 변경 내용은 이 컴퓨터의 브라우저에 저장됩니다. 다른 컴퓨터에서는 변경 내용이 보이지 않을 수 있습니다.
· 사진 파일은 브라우저에 저장되므로 파일 크기가 클수록 느려질 수 있습니다. 2MB 이하를 권장합니다.
⚠️ 위험 구역 — 전체 초기화
모든 변경 내용을 삭제하고 처음 기본값으로 되돌립니다.
이 작업은 취소할 수 없습니다. 신중하게 사용하세요.
