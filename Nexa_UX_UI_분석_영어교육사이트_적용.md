## 1. 디자인 시스템 분석

### 1-1. 컬러 팔레트

| 요소 | 색상 | 코드 |
|------|------|------|
| 배경 | 거의 블랙 | `rgb(13, 13, 13)` |
| 주 텍스트 | 순백 | `rgb(255, 255, 255)` |
| 보조 텍스트 | 반투명 화이트 | `rgba(255, 255, 255, 0.52)` |
| 포인트 (CTA) | 오렌지-레드 | `rgb(247, 82, 27)` |
| 보조 포인트 1 | 블루 | `rgb(27, 170, 247)` |
| 보조 포인트 2 | 그린 | `rgb(27, 247, 89)` |

**핵심 원칙:** 다크 배경 위 고대비 화이트 텍스트. 포인트 컬러는 극도로 절제하여 CTA 버튼과 핵심 강조에만 사용.

### 1-2. 타이포그래피

| 용도 | 폰트 | 특징 |
|------|------|------|
| 헤딩/대형 텍스트 | Sk-Modernist Regular | 산세리프, 기하학적 |
| 라벨/코드 스타일 | Sk-Modernist Mono | 모노스페이스, 기술적 느낌 |
| 본문 | sans-serif (시스템) | 가독성 우선 |

**폰트 사이즈 범위:**
- 최소: 12~14px (라벨, 캡션)
- 본문: 16~23px
- 섹션 헤딩: 69~101px
- 히어로 타이틀: 160~274px (초대형)

**핵심 원칙:** 극단적인 사이즈 대비. 헤딩에 200px+ 대형 타이포를 써서 시각적 임팩트를 만들고, 본문은 16~23px로 읽기 편하게.

### 1-3. 레이아웃 시스템

- **최대 너비:** 1520~1560px (콘텐츠 영역)
- **레이아웃:** Flexbox 100% 사용 (Grid 미사용)
- **주요 간격(Gap):** 12px, 16px, 20px, 32px, 48px, 60px
- **라운딩:** 8px (카드), 12px (입력), 40px (큰 카드), 100px (버튼/뱃지)
- **페이지 전체 높이:** 약 11,000px+ (롱 스크롤 싱글페이지)

## 2. 섹션별 UX 패턴 상세 분석

### 2-1. 내비게이션 바

```
[Nexa*]  PRODUCT  CUSTOMERS  ABOUT  PRICING  BLOG     LOG IN  START FREE TRIAL  [▶ ▌▌▌]
```

**특징:**
- 투명 배경 + 화이트 텍스트 (배경 이미지 위에 떠있는 느낌)
- 좌: 로고 / 중앙: 메뉴 5개 / 우: 로그인 + CTA + 오디오 플레이어(!)
- 오디오 플레이어를 네비게이션에 넣은 것이 독특 — 배경 음악/사운드 디자인 요소
- 스크롤 시에도 상단 고정

**영어교육 적용:**
- `[Isabel English]  COURSES  RESOURCES  ABOUT  BLOG     MY PAGE  START LEARNING`
- 오디오 플레이어 대신 "오늘의 영어 한마디" 미니 위젯 배치 가능

### 2-2. 히어로 섹션

**구성:**
- 풀스크린 시네마틱 이미지 (사무실에서 대화하는 사람들)
- 이미지 위 반투명 오버레이로 가독성 확보
- 하단에 초대형 텍스트: "CUSTOMER SUPPORT AI / TO DRIVE ACTIVATION"
- 좌측 하단: 서브 카피 "For software and scale-ups."
- [SCROLL] 텍스트로 스크롤 유도

**핵심 기법:**
- 텍스트가 화면 하단 1/3에 위치하여 이미지와 겹치면서도 읽힘
- 단어 사이 넓은 자간 → 고급스러운 느낌
- 진입 시 페이드인 애니메이션

**영어교육 적용:**
```
[풀스크린 이미지: 책과 커피가 있는 감각적 공부 장면]

MASTER ENGLISH
    WITH REAL-WORLD CONTENT

"당신만의 영어 학습 여정을 시작하세요."
[SCROLL]
```

## 3. 핵심 디자인 원칙 요약

### 3-1. "Less is More" 극한의 미니멀리즘
- 한 화면에 하나의 메시지만
- 여백을 콘텐츠보다 더 많이 사용
- 색상은 3가지 이내 (블랙, 화이트, 오렌지)

### 3-2. 타이포그래피 = 디자인
- 이미지 대신 초대형 텍스트가 비주얼 역할
- 사이즈 대비가 시각적 위계를 만듦 (12px ↔ 274px)
- 자간과 행간의 넉넉한 여유

### 3-3. 스크롤 스토리텔링
- 위에서 아래로 자연스러운 서사 구조
- 문제 제기 → 솔루션 → 증거(데이터) → 팀 → 가격 → CTA
- 섹션 라벨 [PRODUCT], [PRICING]이 현재 위치 안내

### 3-4. 절제된 인터랙션
- 스크롤 기반 페이드인 애니메이션
- 호버 시 콘텐츠 변경 (팀 섹션)
- 탭 전환으로 콘텐츠 교체 (기능/차트 섹션)
- 화려한 애니메이션 대신 부드러운 트랜지션

### 3-5. 신뢰 구축 패턴
- 데이터 차트로 수치적 증거 제시
- 팀 멤버 실명 + 직책 + 사진
- "Trusted by 10,000+" 소셜 프루프
- 학술 논문 스타일 차트 캡션 (FIG 2.1A)

## 4. 영어 교육 사이트에 적용할 구체적 구조 제안

### 4-1. 추천 페이지 구조 (싱글 페이지)

```
1. 히어로        → 감각적 이미지 + "MASTER ENGLISH" 대형 타이포
2. 가치 제안     → 한 문장 핵심 메시지 + 구분선
3. 강좌 소개     → 인터랙티브 탭 (Speaking/Writing/Grammar) + 미리보기
4. 학습 효과     → 데이터 차트 (수강생 만족도, 실력 향상 그래프)
5. 강사 소개     → 호버형 리스트 (이름 + 전문분야 + 사진)
6. 수강 후기     → 카드 캐러셀 또는 인용문
7. 가격 플랜     → 2~3개 카드 비교
8. CTA           → "지금 시작하세요" + 뉴스레터
9. 푸터          → 네비 + SNS + 대형 브랜드 로고
```

### 4-2. 영어 교육 사이트 추천 컬러 팔레트

**옵션 A: 다크 프리미엄 (Nexa 스타일 유지)**
- 배경: `#0D0D0D` / 텍스트: `#FFFFFF` / 포인트: `#F7521B` (오렌지)

**옵션 B: 따뜻한 다크**
- 배경: `#1A1410` / 텍스트: `#F5F0E8` / 포인트: `#E8A87C` (피치)

**옵션 C: 라이트 모던 (교육 친화적)**
- 배경: `#FAFAF8` / 텍스트: `#1A1A1A` / 포인트: `#2563EB` (블루)

### 4-3. 블로그 콘텐츠 → 웹사이트 콘텐츠 매핑

| 블로그 카테고리 | 웹사이트 섹션 | Nexa 스타일 적용 |
|---------------|-------------|---------------|
| 영어 표현 모음 | Expressions 라이브러리 | 인터랙티브 탭으로 카테고리별 분류 |
| 문법 강의 | Grammar Course | 스크롤 스토리텔링으로 단계별 학습 |
| 영어 뉴스 해설 | Reading Practice | 카드형 아카이브 |
| 회화 팁 | Speaking Guide | 디바이스 목업 안에 대화 시뮬레이션 |
| 시험 준비 | Test Prep | 데이터 차트로 합격률/점수 향상 시각화 |

### 4-4. 추천 기술 스택

- **Framer** — Nexa와 동일. 비개발자도 제작 가능, 애니메이션 내장
- **또는 Next.js + Tailwind CSS + Framer Motion** — 커스터마이징 자유도 높음
- **CMS:** Notion 또는 Contentful (블로그 콘텐츠 관리)

## 5. 체크리스트: 영어 교육 사이트 제작 시 꼭 지킬 것

- [ ] 한 화면에 하나의 메시지만 (여백 과감하게)
- [ ] 헤딩 텍스트 60px 이상으로 시각적 임팩트
- [ ] 포인트 컬러는 CTA 버튼에만 집중 사용
- [ ] 스크롤 기반 자연스러운 콘텐츠 흐름
- [ ] 데이터/수치로 학습 효과 증명 (차트, 통계)
